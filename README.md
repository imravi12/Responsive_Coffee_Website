A fully responsive coffee shop website built using HTML and CSS. This project demonstrates a clean and modern design suitable for small businesses like cafes or restaurants. It features an intuitive user interface, responsive layout, and engaging visual elements.

Features - 
Responsive Design: Works seamlessly across devices (desktop, tablet, and mobile).
Clean Layout: Modern and visually appealing design with a focus on usability.
Interactive Elements: Navigation menu, call-to-action buttons, and animations.
Reusable Code: Well-structured and reusable HTML & CSS code.
